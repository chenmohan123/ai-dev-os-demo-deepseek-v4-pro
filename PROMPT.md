## 一、官网（前台）
响应式设计，必须完美适配 PC 和移动端。
- **首页**：
  - 精美的 Hero 区域，具有现代感的设计，动态或装饰性背景，主标题与行动号召按钮
  - “最新文章”模块，以卡片网格展示，包含封面图、标题、摘要、分类、发布时间
  - 导航栏：Logo、前端页面链接、移动端汉堡菜单，带有毛玻璃效果
  - Footer：版权信息、社交媒体图标（仅展示）
- **文章列表页**：
  - 支持按分类筛选（多级分类以树形或标签形式显示，移动端可折叠）
  - 搜索框，支持前端或后端搜索
  - 分页或无限滚动
- **文章详情页**：
  - 由 Markdown 渲染生成的内容，排版优雅
  - 自动生成面包屑导航（根据分类层级）
  - 文章目录（TOC）在桌面端侧边悬浮，移动端折叠至顶部
  - 社交分享按钮（仅 UI 展示）

## 二、管理后台
需登录才能访问，拥有独立的管理布局（侧边栏 + 顶栏），支持亮色/暗色模式切换。
- **注册/登录页面**：
  - 美观的居中卡片布局，渐变背景，输入框聚焦发光效果
  - 注册后默认分配“普通用户”角色（仅可浏览后台，无管理权限）
- **仪表盘**：
  - 统计卡片：文章总数、用户总数、今日访问量（可用模拟数据）、分类数量
  - 最近更新的文章列表
- **CMS 内容管理**：
  - **文章列表**：表格展示标题、分类、状态（发布/草稿）、作者、更新时间；支持批量删除、搜索、按状态和分类筛选
  - **新建/编辑文章**：
    - 使用 Markdown 编辑器，支持实时预览、全屏、图片插入（图片可存储为 base64 或本地文件）
    - 支持选择**多级分类**（级联选择器，可新建分类或从已有分类树选择）
    - 设置文章状态、SEO 标题、摘要、封面图 URL
    - 自动关联当前登录用户为作者
  - **分类管理**：
    - 无限层级的分类树，支持拖拽排序、编辑、删除
    - 每个分类展示名称、别名、文章数量
- **用户管理**（仅超级管理员可访问）：
  - 表格显示头像、用户名、邮箱、角色、创建时间、状态（启用/禁用）
  - 可新建用户、编辑用户信息、重置密码、分配角色
- **角色管理**（仅超级管理员可访问）：
  - 角色列表，可创建、编辑、删除角色
  - 角色编辑页：通过勾选方式为角色分配权限（权限以树状或分组卡片展示）
- **权限管理**（仅超级管理员可访问）：
  - 权限项列表，支持新增自定义权限标识（如 `article:create`、`article:delete`、`user:manage` 等）
  - 后端和前端均需根据权限标识进行控制：
    - API 层校验权限，无权限返回 403
    - 管理后台中根据角色权限动态显示/隐藏菜单项和操作按钮（可使用类似 `usePermission('article:create')` 的方式进行按钮级控制）

## 三、极致精美的 UI 设计要求
界面必须非常漂亮，现代且富有质感，请融入以下设计要点：
1. **风格**：现代极简主义，融合磨砂玻璃质感（glassmorphism）、柔和阴影和圆角。卡片使用 `rounded-2xl` 或 `rounded-3xl`，带有细腻边框和阴影。
2. **色彩**：亮色模式使用柔和中性色背景与纯白卡片；暗色模式采用深邃的灰蓝黑色系，卡片为半透明深色并带有轻微边框。主色调使用优雅的靛蓝到紫色渐变，按钮、链接、高亮元素均使用该渐变。
3. **动效**：页面切换有淡入淡出或微滑动效果，列表项目交错出现，卡片悬停时轻微上浮并加深阴影，数字滚动动画，骨架屏加载状态。
4. **排版**：标题使用现代无衬线字体，正文字号舒适，行高 1.7～1.8；文章详情排版如 Notion 般美观。
5. **导航**：毛玻璃效果导航栏，背景半透明并带有模糊滤镜，底部 1px 半透明边框。
6. **微交互**：所有可点击元素过渡动画 200-300ms，输入框聚焦时具有柔和的外发光环。
7. **空状态**：使用插画风格的空状态图示（可搭配图标和文字）。
8. **移动端**：官网内容自适应排列，管理后台移动端默认隐藏侧边栏，支持滑出式抽屉。
